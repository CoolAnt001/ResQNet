# 🛡️ ResQNet: Tactical Emergency Bridge

**ResQNet** is a decentralized emergency synchronization system designed for high-stakes rescue scenarios. It provides a robust, "serverless" tactical bridge that synchronizes GPS telemetry, medical profiles, and cyclic video evidence across a cloud-mirrored mesh network.

## 🚀 Key Features
- **Antigravity Swipe SOS**: A high-friction gesture trigger to prevent accidental broadcasts.
- **Biometric Crisis Latch**: A hold-to-secure safety mechanism for arming sensors.
- **Dual-Path Sync**: Mirrors data to local mesh relays and Firebase Cloud for authority access.
- **Cyclic Intel Capture**: Automatically records and uploads 15-second forensic video clips during active distress.
- **Serverless Architecture**: Hosted entirely on GitHub Pages with Firebase real-time persistence.

## 🛠️ Technical Stack
- **Frontend**: Vanilla HTML5, CSS3 (Tactical Dark Mode), Javascript (ES6)
- **Database**: Firebase Firestore (Real-time SOS logs)
- **Storage**: Firebase Cloud Storage (Video evidence)
- **Networking**: P2P Local Polling + Web Bluetooth (Experimental)

## 📡 Deployment
This repository is optimized for **GitHub Pages**. To deploy:
1. Enable GitHub Pages in the repository settings.
2. The app will be live at `https://coolant001.github.io/ResQNet/`.
